清洗代码示例

# 自动数据清洗流程 def load_sales_data(uploaded_file): df = pd.read_csv(uploaded_file) # 1. 列名自动映射（中文→英文） column_mapping = { '订单日期': 'order_date', '订单金额': 'revenue', '订单成本': 'cost', '商品类目': 'category', '客户ID': 'customer_id', '城市': 'city', '渠道': 'channel' } df = df.rename(columns=column_mapping) # 2. 数据类型转换 df['order_date'] = pd.to_datetime(df['order_date']) # 3. 自动计算利润 if 'profit' not in df.columns: df['profit'] = df['revenue'] - df['cost'] # 4. 生成月份字段 df['order_month'] = df['order_date'].values.astype('datetime64[M]') return df

✅ 清洗结果

  • 原始数据：5,000 条订单记录
  • 去重后：4,985 条有效记录（15条重复已清理）
  • 缺失值：自动填充 23 个缺失值
  • 日期格式：统一转换为 datetime 类型
  • 利润计算：自动计算 100% 记录

🚀 快速开始

1. 安装依赖

git clone https://github.com/keven798/nexustik.git cd nexustik pip install -r requirements.txt

2. 启动平台

cd nexus-platform streamlit run app.py

3. 导入数据

# 在网页界面中操作： # 1. 选择"销售分析"或"客户分析"页面 # 2. 点击"上传CSV文件" # 3. 选择从抖音导出的数据文件 # 4. 系统自动清洗并生成看板
