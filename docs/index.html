客户分析

📱

视频数据分析、直播效果追踪、互动率统计，全面掌握抖音电商表现

中文/English 一键切换，满足不同语言环境需求

🎨

CSV导入

分析模块

📊 销售分析模块

  • 销售趋势折线图（月度/季度）
  • 类目分布柱状图
  • 渠道占比饼图
  • 6大KPI指标卡片
  • 多维度数据筛选
  • CSV数据导入支持

👥 客户分析模块

  • RFM客户分群分析
  • 留存分析热力图
  • 客户价值评估
  • 新客户/活跃/高价值分布
  • 城市/类目筛选
  • CSV数据导入支持

📱 抖音分析模块

  • 视频数据趋势图
  • 直播效果散点图
  • 播放量/点赞/评论统计
  • 互动率分析
  • GMV/转化率追踪
  • 视频/直播KPI对比

快速开始

1. 安装依赖

git clone https://github.com/yourusername/nexustik-analytics.git
cd nexustik-analytics
pip install -r requirements.txt

2. 启动统一分析平台

cd nexus-platform
streamlit run app.py
# 访问 http://localhost:8501

3. 启动销售智能看板

cd sales-intelligence-dashboard
python app.py
# 访问 http://localhost:8050

4. 启动客户分析看板

cd customer-analytics-dashboard
streamlit run app.py
# 访问 http://localhost:8502

5. 启动抖音视频看板

cd douyin-video-dashboard
npm install
npm run dev
# 访问 http://localhost:5173
